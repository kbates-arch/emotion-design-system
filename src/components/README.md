This directory contains the design system component buckets.

Atomic placement rules:

- Place basic building blocks in `atoms` (e.g. `Button`, `Input`).
- Place composed components in `molecules` (e.g. `SearchField`).
- Place larger, page-level sections in `organisms` (e.g. `Header`).

Follow the BEM naming convention for CSS classes and prefer semantic HTML for interactive elements.

Refer to the repository agent rules: `.github/agents/design-systems-architect-expert-agent.agent.md` for placement and rules.
