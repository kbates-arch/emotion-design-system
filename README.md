# Velvet UI – Bakery Design System

A reusable bakery-themed design system built with React and TypeScript. Velvet UI documents colors, typography, components, interaction states, and usage guidelines for modern bakery interfaces and food-focused digital experiences.

## Features

* Light and dark mode
* Reusable UI components
* Interactive component previews
* Component states and variations
* Design token documentation
* Usage guidelines with Do / Don’t examples
* Responsive layout
* Clean bakery-inspired visual identity

## Components Included

* Buttons
* Badges
* Inputs
* Product Cards
* Navigation
* Alerts
* Color Tokens
* Typography
* Toggles

## Built With

* React
* TypeScript
* CSS
* Vite

## Design Style

Velvet UI uses a warm bakery-inspired aesthetic with:

* Soft cream backgrounds
* Caramel and cocoa accent colors
* Rounded cards and components
* Elegant typography
* Minimal and clean layouts

## Component States

The system documents multiple interaction states including:

* Default
* Hover
* Active
* Disabled
* Selected
* Success
* Error
* Warning

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Purpose

This project was created as a UI design system exploration focused on bakery branding and reusable interface patterns. The goal was to create a scalable visual system that documents component behavior, states, spacing, color usage, and accessibility considerations.
