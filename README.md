# Velvet & Crumbs — Bakery Landing

This repository now contains a simple bakery landing page demo (React + Vite). The previous design-system demo has been removed and the app now renders a single-page bakery site with header, hero, about, features, popular treats, and footer.

Quick links:
- App entry: [src/App.tsx](src/App.tsx#L1-L1)
- Styles: [src/App.css](src/App.css#L1-L1)

Development:

```bash
npm install
npm run dev
```

Notes:
- The site uses soft cream and pink tones, Playfair Display for headings, and Inter for body text.
- Images are sourced from Unsplash placeholders.

If you want the old design-system files fully purged (including compiled `dist` assets), I can remove them next — tell me if you'd like that.
